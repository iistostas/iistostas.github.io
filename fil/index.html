Browser Fingerprint Checker

Pindutin ang button para basahin ang mga signal ng fingerprint na tahimik na inilalantad ng iyong browser - canvas, WebGL, naka-install na font, time zone, screen at User-Agent - pagkatapos ay makakuha ng pinagsamang fingerprint ID at tingnan kung gaano kadaling matukoy ka ng bawat signal. Lahat ay kinakalkula nang lokal sa iyong browser; walang data na ina-upload.

Tumatakbo ang tool na ito nang buo sa iyong browser. Walang data na ina-upload o iniimbak.

Mga signal na bumubuo ng browser fingerprint

Hindi kailangan ng cookie ng isang site: mula sa ilang tila walang-saysay na tanong na sinasagot ng browser, makakabuo ito ng medyo matatag na pantukoy. Kabilang sa karaniwang signal ang banayad na pagkakaiba ng GPU at driver kapag nagre-render ng canvas/WebGL, listahan ng naka-install na font, time zone, resolution ng screen at lalim ng kulay, wika, bilang ng CPU core, memory ng device at suporta sa touch. Walang iisang halaga na natatangi, ngunit kapag pinagsama ay kadalasang nagiging natatangi.

Gaano nakakatukoy ang bawat signal (entropy)

Sinusukat ng entropy kung gaano karaming impormasyong nakakatukoy ang dala ng isang signal (sa bit - mas mataas, mas naihihiwalay ang mga tao). Sinukat ng Panopticlick ng EFF ang average na humigit-kumulang 18.1 bits, na may humigit-kumulang 83.6% na natatanging fingerprint; natuklasan ng AmIUnique (Laperdrix et al., 2016) na humigit-kumulang 89.4% ang natatangi sa 118,000 sample. Nagbibigay ang talahanayan sa ibaba ng magaspang na tantiya kada signal na hinango sa pampublikong pananaliksik na iyon.

Entropy ng fingerprint kada signal (tantiya 2026)

Ang mga sumusunod ay tantiya para sa 2026 na hinango sa pampublikong pananaliksik sa fingerprint (EFF Panopticlick 2010, AmIUnique 2016), na nagpapakita ng tinatayang impormasyong nakakatukoy kada signal; tantiya, hindi direktang sukat, para sa sanggunian lamang.

Mga madalas itanong

Ano ang browser fingerprint?

Ang browser fingerprint ay isang pantukoy na binuo sa pamamagitan ng pagsasama ng mga katangiang inilalantad ng iyong browser at device (user agent, screen, time zone, font, canvas/WebGL rendering at iba pa). Kahit walang cookie, makikilala ng site ang parehong browser sa iba't ibang pagbisita.

Ligtas ba ang checker na ito? Ina-upload ba nito ang aking data?

Hindi. Lahat ng pagbabasa at pagkalkula ay nangyayari nang lokal sa browser gamit ang JavaScript; ipinapakita ang resulta sa pahinang ito lamang at hindi kailanman ipinapadala sa anumang server o iniimbak.

Bakit natatangi ang aking fingerprint?

Ang iisang katangian (tulad ng time zone) ay magkapareho sa maraming tao, ngunit ang dose-dosenang katangiang pinagsama ay lumilikha ng napakalaking espasyo ng halaga. Ipinapakita ng pampublikong pananaliksik na karamihan sa browser ay may natatanging pinagsamang fingerprint, kaya matutunton sa iba't ibang site nang hindi nag-lo-log in.

Ano ang canvas fingerprint?

Gumuguhit ang browser ng parehong teksto o hugis sa nakatagong canvas at babasahin muli ang mga pixel. Ang magkakaibang GPU, driver, system font at anti-aliasing ay lumilikha ng maliit ngunit matatag na pagkakaiba, kaya isa ang canvas sa pinakanakakatukoy na signal.

Itinatago ba ng incognito o private mode ang fingerprint?

Kadalasan ay hindi. Nililinis ng private mode ang cookie at lokal na kasaysayan, ngunit nananatiling nakalantad ang canvas, WebGL, font, time zone at mga katulad na signal, kaya hindi sapat ang private mode lamang para makaiwas sa fingerprint tracking.

Paano babawasan ang fingerprint tracking?

Maaari mong paliitin ang pagkalantad (gumamit ng karaniwang setting, i-disable ang ilang API) o gumamit ng anti-detect browser para sa maraming account - nagtatalaga ito sa bawat profile ng hiwalay, makokontrol at magkatugmang fingerprint para hindi maiugnay ang mga account.

Mababago ba ng VPN ang aking fingerprint?

Hindi. Pinapalitan lang ng VPN ang exit IP; hindi nito binabago ang canvas, font, time zone o user-agent, kaya pareho pa rin ang fingerprint pagkatapos palitan ang IP.
