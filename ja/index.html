ブラウザフィンガープリント検査

ボタンを押すと、ブラウザが知らないうちに公開しているフィンガープリント信号（canvas・WebGL・インストール済みフォント・タイムゾーン・画面・User-Agent）を読み取り、合成フィンガープリント ID を計算し、各信号がどれだけあなたを特定しやすいかを表示します。計算はすべてブラウザ内で行われ、データは一切送信されません。

本ツールは完全にブラウザ内で動作し、データの送信・保存は行いません。

ブラウザフィンガープリントを構成する信号

サイトは cookie がなくても、ブラウザが答える一見「無害」な質問を集めるだけで、かなり安定した識別子を組み立てられます。代表的な要素は、canvas/WebGL 描画時の GPU・ドライバの微差、インストール済みフォント一覧、タイムゾーン、画面解像度と色深度、言語、CPU コア数、デバイスメモリ、タッチ対応など。単独では一意でなくても、組み合わせるとたいてい一意になります。

各信号の特定力（エントロピー）

エントロピーは信号がどれだけ識別情報を持つかを示します（単位 bit・高いほど人を区別できる）。EFF の Panopticlick は平均約 18.1 bit、約 83.6% のフィンガープリントが一意と測定。AmIUnique（Laperdrix 他, 2016）は 11.8 万件で約 89.4% が一意でした。下表はこれら公開研究をまとめた大まかな目安です。

よくある質問

ブラウザフィンガープリントとは？

ブラウザとデバイスが公開する属性（User-Agent、画面、タイムゾーン、フォント、canvas/WebGL 描画など）を組み合わせて作る識別子です。cookie がなくても、同じブラウザを訪問をまたいで認識できます。

この検査は安全？データは送信される？

送信されません。読み取りと計算はすべてブラウザ内の JavaScript で行われ、結果は本ページに表示されるだけで、サーバーへの送信も保存もしません。

なぜ私のフィンガープリントは一意なの？

単一の属性（例：タイムゾーン）は多くの人と同じでも、十数個の属性を組み合わせると値の空間が膨大になります。公開研究では大半のブラウザの組み合わせが一意で、ログインなしでもサイト横断で追跡され得ます。

canvas フィンガープリントとは？

同じ文字や図形を隠し canvas に描かせ、ピクセルを読み戻します。GPU・ドライバ・システムフォント・アンチエイリアスの違いで微妙だが安定した差が出るため、canvas は特定力が非常に高い信号です。

シークレット／プライベートモードで隠せる？

ほぼ隠せません。プライベートモードは主に cookie と履歴を消しますが、canvas・WebGL・フォント・タイムゾーンなどの信号はそのまま公開されるため、それだけでは追跡を逃れられません。

フィンガープリント追跡を減らすには？

露出を減らす（一般的な設定にそろえる、一部 API を無効化）か、複数アカウント向けのフィンガープリントブラウザを使います。各プロファイルに独立かつ制御可能で整合した指紋を割り当て、アカウント同士をひも付けません。

VPN でフィンガープリントは変わる？

変わりません。VPN は出口 IP を変えるだけで、canvas・フォント・タイムゾーン・User-Agent などブラウザ側の信号は変えないため、IP を変えても指紋は同じです。
