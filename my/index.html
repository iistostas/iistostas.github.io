ဘရောက်ဇာ လက်ဗွေ စစ်ဆေးကိရိယာ

ခလုတ်ကို နှိပ်၍ သင့်ဘရောက်ဇာက တိတ်တဆိတ် ဖော်ထုတ်နေသော လက်ဗွေ အချက်ပြများ — canvas, WebGL, တပ်ဆင်ထားသော ဖောင့်, အချိန်ဇုန်, မျက်နှာပြင် နှင့် User-Agent — ကို ဖတ်ပြီး ပေါင်းစပ်လက်ဗွေ ID ကို တွက်ချက်ကာ အချက်ပြတစ်ခုစီက သင့်ကို မည်မျှ လွယ်ကူစွာ ခွဲခြားနိုင်သည်ကို ကြည့်ပါ။ အရာအားလုံးကို သင့်ဘရောက်ဇာတွင် စက်တွင်း တွက်ချက်ပြီး ဒေတာ မတင်ပါ။

ဤကိရိယာသည် သင့်ဘရောက်ဇာတွင် အပြည့်အဝ လည်ပတ်သည်။ ဒေတာ မတင်၊ မသိမ်းပါ။

ဘရောက်ဇာ လက်ဗွေကို ဖွဲ့စည်းသော အချက်ပြများ

ဝဘ်ဆိုက်သည် cookie မလို: ဘရောက်ဇာ ဖြေဆိုသော အန္တရာယ်မရှိဟု ထင်ရသော မေးခွန်းအနည်းငယ်မှပင် တော်တော်တည်ငြိမ်သော အမှတ်အသားတစ်ခု တည်ဆောက်နိုင်သည်။ အသုံးများသော အချက်ပြများတွင် canvas/WebGL ရင်ဒါလုပ်စဉ် GPU နှင့် driver ၏ သေးငယ်သော ကွာခြားမှု, တပ်ဆင်ထားသော ဖောင့်စာရင်း, အချိန်ဇုန်, မျက်နှာပြင် resolution နှင့် အရောင်အနက်, ဘာသာစကား, CPU core အရေအတွက်, စက်ပစ္စည်း memory နှင့် touch ပံ့ပိုးမှု ပါဝင်သည်။ တန်ဖိုးတစ်ခုချင်းသည် ထူးခြားမှု မရှိသော်လည်း ပေါင်းစပ်လိုက်သောအခါ များသောအားဖြင့် ထူးခြားသွားသည်။

အချက်ပြတစ်ခုစီ မည်မျှ ခွဲခြားနိုင်သနည်း (entropy)

Entropy သည် အချက်ပြတစ်ခုက ခွဲခြားသိနိုင်သော သတင်းအချက်အလက် မည်မျှ သယ်ဆောင်သည်ကို တိုင်းတာသည် (bit ဖြင့် — မြင့်လေ လူခွဲခြားနိုင်လေ)။ EFF ၏ Panopticlick က ပျမ်းမျှ 18.1 bit ခန့်၊ လက်ဗွေ 83.6% ခန့် ထူးခြားသည်ဟု တိုင်းတာသည်။ AmIUnique (Laperdrix et al., 2016) က နမူနာ 118,000 တွင် 89.4% ခန့် ထူးခြားသည်ကို တွေ့သည်။ အောက်ပါဇယားသည် ထို အများသုံးသုတေသနမှ စုစည်းထားသော အချက်ပြအလိုက် ကြမ်းကြမ်း ခန့်မှန်းချက် ဖြစ်သည်။

အချက်ပြအလိုက် လက်ဗွေ entropy (2026 ခန့်မှန်းချက်)

အောက်ပါတို့သည် အများသုံး လက်ဗွေ သုတေသန (EFF Panopticlick 2010, AmIUnique 2016) မှ စုစည်းထားသော 2026 ခန့်မှန်းချက် ဖြစ်ပြီး အချက်ပြအလိုက် ခွဲခြားသိနိုင်သော သတင်းအချက်အလက် ခန့်မှန်းကို ပြသသည်။ ခန့်မှန်းချက်၊ တိုက်ရိုက်တိုင်းတာမှု မဟုတ်၊ ကိုးကားရန်သာ။

မေးလေ့ရှိသော မေးခွန်းများ

ဘရောက်ဇာ လက်ဗွေ ဆိုသည်မှာ အဘယ်နည်း?

ဘရောက်ဇာ လက်ဗွေသည် သင့်ဘရောက်ဇာနှင့် စက်ပစ္စည်းက ဖော်ထုတ်သော အရည်အသွေးများ (user agent, မျက်နှာပြင်, အချိန်ဇုန်, ဖောင့်, canvas/WebGL ရင်ဒါ စသည်) ကို ပေါင်းစပ်၍ တည်ဆောက်သော အမှတ်အသား ဖြစ်သည်။ cookie မရှိသော်လည်း ဝဘ်ဆိုက်က တူညီသော ဘရောက်ဇာကို လည်ပတ်မှုများတစ်လျှောက် မှတ်မိနိုင်သည်။

ဤစစ်ဆေးကိရိယာ လုံခြုံပါသလား? ကျွန်ုပ်ဒေတာ တင်ပါသလား?

မတင်ပါ။ ဖတ်ခြင်းနှင့် တွက်ချက်ခြင်း အားလုံးကို ဘရောက်ဇာတွင် JavaScript ဖြင့် စက်တွင်း ပြုလုပ်သည်။ ရလဒ်ကို ဤစာမျက်နှာတွင်သာ ပြသပြီး မည်သည့် ဆာဗာသို့မျှ မပို့၊ မသိမ်းပါ။

ကျွန်ုပ်၏ လက်ဗွေသည် အဘယ်ကြောင့် ထူးခြားသနည်း?

အရည်အသွေးတစ်ခုတည်း (အချိန်ဇုန်ကဲ့သို့) ကို လူများစွာ မျှဝေသော်လည်း အရည်အသွေး ဆယ်စုကို ပေါင်းစပ်လိုက်လျှင် တန်ဖိုးနေရာ အလွန်ကြီးမားသွားသည်။ အများသုံးသုတေသနက ဘရောက်ဇာ အများစုတွင် ထူးခြားသော ပေါင်းစပ်လက်ဗွေ ရှိသည်ဟု ပြသ၍ login မလုပ်ဘဲ ဆိုက်များဖြတ်၍ ခြေရာခံနိုင်သည်။

canvas လက်ဗွေ ဆိုသည်မှာ အဘယ်နည်း?

ဘရောက်ဇာက တူညီသော စာသား သို့မဟုတ် ပုံကို ဖုံးကွယ်ထားသော canvas တွင် ဆွဲပြီး pixel များကို ပြန်ဖတ်သည်။ မတူသော GPU, driver, စနစ်ဖောင့် နှင့် anti-aliasing က သေးငယ်သော်လည်း တည်ငြိမ်သော ကွာခြားမှု ဖြစ်စေ၍ canvas သည် အခွဲခြားနိုင်ဆုံး အချက်ပြများထဲမှ တစ်ခု ဖြစ်သည်။

incognito mode က လက်ဗွေကို ဖုံးကွယ်နိုင်ပါသလား?

အများအားဖြင့် မဖုံးကွယ်နိုင်ပါ။ private mode သည် cookie နှင့် စက်တွင်း မှတ်တမ်းကို အဓိက ရှင်းသော်လည်း canvas, WebGL, ဖောင့်, အချိန်ဇုန် စသော အချက်ပြများ ဆက်ဖော်ထုတ်နေသဖြင့် private mode တစ်ခုတည်းဖြင့် လက်ဗွေ ခြေရာခံမှုကို မလွတ်နိုင်ပါ။

လက်ဗွေ ခြေရာခံမှုကို မည်သို့ လျှော့ချမည်နည်း?

ဖော်ထုတ်မှုကို လျှော့ချနိုင်သည် (အသုံးများသော settings သုံး၊ အချို့ API ပိတ်) သို့မဟုတ် အကောင့်များအတွက် anti-detect ဘရောက်ဇာ သုံးနိုင်သည် — profile တစ်ခုစီကို သီးခြား၊ ထိန်းချုပ်နိုင်ပြီး ကိုက်ညီသော လက်ဗွေ သတ်မှတ်ပေး၍ အကောင့်များ မချိတ်ဆက်စေရန် ဖြစ်သည်။

VPN က ကျွန်ုပ်၏ လက်ဗွေကို ပြောင်းနိုင်ပါသလား?

မပြောင်းနိုင်ပါ။ VPN သည် ထွက်ပေါက် IP ကိုသာ ပြောင်းပြီး canvas, ဖောင့်, အချိန်ဇုန် သို့မဟုတ် user-agent ကို မပြောင်းသဖြင့် IP ပြောင်းပြီးနောက် လက်ဗွေ မပြောင်းပါ။
