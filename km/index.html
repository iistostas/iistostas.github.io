бһ§бһ”бһҖбһҡбһҺбҹҚбһ“бҹҒбҹҮбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҳбһ·бһ“бһ”бһүбҹ’бһҮбһјбһ“ бһ¬бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһЎбһҫбһҷбҹ”

бһҹбһүбҹ’бһүбһ¶бһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹ’бһ“бһ¶бһҳбһҳбҹ’бһҡбһ¶бһҳбһҠбҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ

бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ cookie бһЎбһҫбһҷбҹ– бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ”бҹ’бһҡбһҳбһјбһӣбһ…бһҳбҹ’бһӣбһҫбһҷбһҠбҹӮбһӣбһҳбһҫбһӣбһ‘бҹ…бһӮбҹ’бһҳбһ¶бһ“бһӮбҹ’бһҡбҹ„бҹҮбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ–бһёбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ бһҖбҹҸбһўбһ¶бһ…бһ•бҹ’бһӮбһ»бҹҶбһҮбһ¶бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҠбҹҸбһҹбҹ’бһҗбһ·бһҸбһҹбҹ’бһҗбҹҒбһҡбһ”бһ¶бһ“бҹ” бһҹбһүбҹ’бһүбһ¶бһ‘бһјбһ‘бҹ…бһҡбһҪбһҳбһҳбһ¶бһ“бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҸбһ·бһ…бһҸбһҪбһ…бһ“бҹғ GPU бһ“бһ·бһ„ driver бһ–бҹҒбһӣбһ”бһ„бҹ’бһ бһ¶бһү canvas/WebGL, бһ”бһүбҹ’бһҮбһёбһ–бһ»бһҳбҹ’бһ–бһўбһҖбҹ’бһҹбһҡбһҠбҹӮбһӣбһҠбҹҶбһЎбһҫбһ„, бһҸбҹҶбһ”бһ“бҹӢбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶, бһӮбһ»бһҺбһ—бһ¶бһ–бһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ бһ“бһ·бһ„бһҮбһҳбҹ’бһҡбҹ…бһ–бһҺбҹҢ, бһ—бһ¶бһҹбһ¶, бһ…бҹҶбһ“бһҪбһ“бһҹбҹ’бһ“бһјбһӣ CPU, бһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶбһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһ”бҹүбҹҮбҹ” бһҸбһҳбҹ’бһӣбҹғбһ“бһёбһҳбһҪбһҷбҹ—бһҳбһ·бһ“бһҳбҹӮбһ“бһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ–бҹҒбһӣбһ•бҹ’бһҹбҹҶбһӮбҹ’бһ“бһ¶бһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶бһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢбҹ”

бһҹбһүбҹ’бһүбһ¶бһ“бһёбһҳбһҪбһҷбҹ—бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһ”бһ¶бһ“бһҒбҹ’бһӣбһ¶бҹҶбһ„бһ”бҹүбһ»бһҺбҹ’бһҺбһ¶ (entropy)

Entropy бһңбһ¶бһҹбҹӢбһ”бһҡбһ·бһҳбһ¶бһҺбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҠбҹӮбһӣбһҹбһүбҹ’бһүбһ¶бһҳбһҪбһҷбһ•бҹ’бһ‘бһ»бһҖ (бһҮбһ¶ bit вҖ” бһҖбһ¶бһ“бҹӢбһҸбҹӮбһҒбҹ’бһ–бһҹбҹӢ бһҖбһ¶бһ“бҹӢбһҸбҹӮбһүбҹӮбһҖбһҳбһ“бһ»бһҹбҹ’бһҹбһ”бһ¶бһ“)бҹ” Panopticlick бһҡбһ”бһҹбҹӢ EFF бһңбһ¶бһҹбҹӢбһ”бһ¶бһ“бһҮбһ¶бһҳбһ’бҹ’бһҷбһҳбһ”бҹ’бһҡбһ бҹӮбһӣ 18.1 bit бһҠбҹ„бһҷбһҹбҹ’бһ“бһ¶бһҳбһҳбҹ’бһҡбһ¶бһҳбһҠбҹғбһ”бҹ’бһҡбһ бҹӮбһӣ 83.6% бһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢ бһҡбһёбһҜ AmIUnique (Laperdrix et al., 2016) бһҡбһҖбһғбһҫбһүбһ”бҹ’бһҡбһ бҹӮбһӣ 89.4% бһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢбһҖбҹ’бһ“бһ»бһ„бһӮбҹҶбһҡбһј 118,000бҹ” бһҸбһ¶бһҡбһ¶бһ„бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ•бҹ’бһҠбһӣбҹӢбһҖбһ¶бһҡбһ”бҹүбһ¶бһ“бҹӢбһҹбҹ’бһҳбһ¶бһ“бһ”бҹ’бһҡбһ бһ¶бһҖбҹӢбһ”бҹ’бһҡбһ бҹӮбһӣбһҖбҹ’бһ“бһ»бһ„бһҹбһүбҹ’бһүбһ¶бһ“бһёбһҳбһҪбһҷбҹ—бһҠбҹӮбһӣбһҹбһҡбһ»бһ”бһ–бһёбһҖбһ¶бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһҹбһ¶бһ’бһ¶бһҡбһҺбҹҲбһ“бҹ„бҹҮбҹ”

Entropy бһҹбҹ’бһ“бһ¶бһҳбһҳбҹ’бһҡбһ¶бһҳбһҠбҹғбһҸбһ¶бһҳбһҹбһүбҹ’бһүбһ¶ (бһҖбһ¶бһҡбһ”бҹүбһ¶бһ“бҹӢбһҹбҹ’бһҳбһ¶бһ“ 2026)

бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮбһҮбһ¶бһҖбһ¶бһҡбһ”бҹүбһ¶бһ“бҹӢбһҹбҹ’бһҳбһ¶бһ“бһҶбҹ’бһ“бһ¶бҹҶ 2026 бһҠбҹӮбһӣбһҹбһҡбһ»бһ”бһ–бһёбһҖбһ¶бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһҹбҹ’бһ“бһ¶бһҳбһҳбҹ’бһҡбһ¶бһҳбһҠбҹғбһҹбһ¶бһ’бһ¶бһҡбһҺбҹҲ (EFF Panopticlick 2010, AmIUnique 2016) бһ”бһ„бҹ’бһ бһ¶бһүбһ”бһҡбһ·бһҳбһ¶бһҺбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһ”бҹ’бһҡбһ бһ¶бһҖбҹӢбһ”бҹ’бһҡбһ бҹӮбһӣбһҖбҹ’бһ“бһ»бһ„бһҹбһүбҹ’бһүбһ¶бһ“бһёбһҳбһҪбһҷбҹ—бҹ– бһҖбһ¶бһҡбһ”бҹүбһ¶бһ“бҹӢбһҹбҹ’бһҳбһ¶бһ“ бһҳбһ·бһ“бһҳбҹӮбһ“бһҖбһ¶бһҡбһңбһ¶бһҹбҹӢбһ•бҹ’бһ‘бһ¶бһӣбҹӢ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҷбҹ„бһ„бһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ”

бһҹбҹ’бһ“бһ¶бһҳбһҳбҹ’бһҡбһ¶бһҳбһҠбҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҮбһ¶бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷбһ•бҹ’бһҹбҹҶбһӣбһҖбҹ’бһҒбһҺбҹҲбһҠбҹӮбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһ бһ¶бһү (user agent, бһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ, бһҸбҹҶбһ”бһ“бҹӢбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶, бһ–бһ»бһҳбҹ’бһ–бһўбһҖбҹ’бһҹбһҡ, бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһү canvas/WebGL бҹ”бһӣбҹ”)бҹ” бһ‘бҹ„бҹҮбһӮбҹ’бһҳбһ¶бһ“ cookie бһҖбҹҸбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһўбһ¶бһ…бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҠбһҠбҹӮбһӣбһҶбҹ’бһӣбһ„бһҖбһ¶бһҡбһ…бһјбһӣбһҳбһҫбһӣбһ”бһ¶бһ“бҹ”

бһ§бһ”бһҖбһҡбһҺбҹҚбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ“бҹҒбҹҮбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ‘бҹҒ? бһҸбһҫбһңбһ¶бһ”бһүбҹ’бһҮбһјбһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһҒбҹ’бһүбһ»бҹҶбһ‘бҹҒ?

бһ‘бҹҒбҹ” бһҖбһ¶бһҡбһўбһ¶бһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбһҺбһ“бһ¶бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҖбһҫбһҸбһЎбһҫбһ„бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҠбҹ„бһҷ JavaScript бһӣбһ‘бҹ’бһ’бһ•бһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҸбҹӮбһӣбһҫбһ‘бҹҶбһ–бҹҗбһҡбһ“бҹҒбҹҮ бһҳбһ·бһ“бһ”бһүбҹ’бһҮбһјбһ“бһ‘бҹ…бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһҺбһ¶бһҳбһҪбһҷ бһ¬бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһЎбһҫбһҷбҹ”

бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһҹбҹ’бһ“бһ¶бһҳбһҳбҹ’бһҡбһ¶бһҳбһҠбҹғбһҡбһ”бһҹбҹӢбһҒбҹ’бһүбһ»бҹҶбһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢ?

бһӣбһҖбҹ’бһҒбһҺбҹҲбһҸбҹӮбһҳбһҪбһҷ (бһҠбһјбһ…бһҮбһ¶бһҸбҹҶбһ”бһ“бҹӢбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶) бһҳбһ¶бһ“бһҳбһ“бһ»бһҹбҹ’бһҹбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ…бҹ’бһҡбһҫбһ“ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһӣбһҖбҹ’бһҒбһҺбҹҲбһҡбһ¶бһ”бҹӢбһҹбһ·бһ”бһ•бҹ’бһҹбҹҶбһӮбҹ’бһ“бһ¶бһ”бһ„бҹ’бһҖбһҫбһҸбһӣбҹҶбһ бһҸбһҳбҹ’бһӣбҹғбһ’бҹҶбһҹбһҳбҹ’бһ”бһҫбһҳбҹ” бһҖбһ¶бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһҹбһ¶бһ’бһ¶бһҡбһҺбҹҲбһ”бһ„бҹ’бһ бһ¶бһүбһҗбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһҳбһ¶бһ“бһҹбҹ’бһ“бһ¶бһҳбһҳбҹ’бһҡбһ¶бһҳбһҠбҹғбһҡбһҪбһҳбһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһҸбһ¶бһҳбһҠбһ¶бһ“бһҶбҹ’бһӣбһ„бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҠбҹ„бһҷбһҳбһ·бһ“бһ”бһ¶бһ…бҹӢбһ…бһјбһӣбһӮбһҺбһ“бһёбҹ”

бһҹбҹ’бһ“бһ¶бһҳбһҳбҹ’бһҡбһ¶бһҳбһҠбҹғ canvas бһҮбһ¶бһўбҹ’бһңбһё?

бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһӮбһјбһҡбһўбһҖбҹ’бһҹбһҡ бһ¬бһҡбһјбһ”бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ canvas бһӣбһ¶бһҖбҹӢ бһҡбһҪбһ…бһўбһ¶бһ“ pixel бһҸбҹ’бһҡбһЎбһ”бҹӢбһҳбһҖбһңбһ·бһүбҹ” GPU, driver, бһ–бһ»бһҳбҹ’бһ–бһўбһҖбҹ’бһҹбһҡбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„ anti-aliasing бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ”бһ„бҹ’бһҖбһҫбһҸбһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҸбһ·бһ…бһҸбҹӮбһҹбҹ’бһҗбһ·бһҸбһҹбҹ’бһҗбҹҒбһҡ бһ’бҹ’бһңбһҫбһұбҹ’бһҷ canvas бһҮбһ¶бһҹбһүбҹ’бһүбһ¶бһҠбҹӮбһӣбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҒбҹ’бһӣбһ¶бҹҶбһ„бһ”бҹҶбһ•бһ»бһҸбһҳбһҪбһҷбҹ”

бһҡбһ”бҹҖбһ”бһўбһ“бһ¶бһҳбһ·бһҖ (incognito) бһӣбһ¶бһҖбҹӢбһҹбҹ’бһ“бһ¶бһҳбһҳбҹ’бһҡбһ¶бһҳбһҠбҹғбһ”бһ¶бһ“бһ‘бҹҒ?

бһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһҳбһ·бһ“бһ”бһ¶бһ“бҹ” бһҡбһ”бҹҖбһ”бһҜбһҖбһҮбһ“бһӣбһ»бһ” cookie бһ“бһ·бһ„бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҮбһ¶бһ…бһҳбҹ’бһ”бһ„ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮ canvas, WebGL, бһ–бһ»бһҳбҹ’бһ–бһўбһҖбҹ’бһҹбһҡ, бһҸбҹҶбһ”бһ“бҹӢбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶ бһ“бҹ…бһҸбҹӮбһ”бһ„бҹ’бһ бһ¶бһү бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҡбһ”бҹҖбһ”бһҜбһҖбһҮбһ“бһҸбҹӮбһҜбһ„бһҳбһ·бһ“бһўбһ¶бһ…бһӮбҹҒбһ…бһ–бһёбһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“бһҹбҹ’бһ“бһ¶бһҳбһҳбҹ’бһҡбһ¶бһҳбһҠбҹғбһ”бһ¶бһ“бһ‘бҹҒбҹ”

бһҸбһҫбһ’бҹ’бһңбһҫбһҠбһјбһ…бһҳбҹ’бһҠбҹҒбһ…бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“бһҹбҹ’бһ“бһ¶бһҳбһҳбҹ’бһҡбһ¶бһҳбһҠбҹғ?

бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ“бҹ’бһҗбһҷбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһү (бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ‘бһјбһ‘бҹ… бһ”бһ·бһ‘ API бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“) бһ¬бһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһ”бҹ’бһҡбһҶбһ¶бҹҶбһ„бһҖбһ¶бһҡбһҡбһҖбһғбһҫбһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһҺбһ“бһёбһ…бҹ’бһҡбһҫбһ“ вҖ” бһңбһ¶бһ•бҹ’бһҠбһӣбҹӢбһҹбҹ’бһ“бһ¶бһҳбһҳбҹ’бһҡбһ¶бһҳбһҠбҹғбһҠбһ¶бһ…бҹӢбһҠбҹ„бһҷбһЎбҹӮбһҖ бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһ¶бһ“ бһ“бһ·бһ„бһҹбҹҠбһёбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢбһӮбҹ’бһ“бһ¶бһҠбһӣбҹӢ profile бһ“бһёбһҳбһҪбһҷбҹ— бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ»бҹҶбһұбҹ’бһҷбһӮбһҺбһ“бһёбһ—бҹ’бһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶бҹ”

VPN бһўбһ¶бһ…бһ”бҹ’бһҠбһјбһҡбһҹбҹ’бһ“бһ¶бһҳбһҳбҹ’бһҡбһ¶бһҳбһҠбҹғбһҡбһ”бһҹбҹӢбһҒбҹ’бһүбһ»бҹҶбһ”бһ¶бһ“бһ‘бҹҒ?

бһҳбһ·бһ“бһ”бһ¶бһ“бһ‘бҹҒбҹ” VPN бһ”бҹ’бһҠбһјбһҡбһҸбҹӮ IP бһ…бҹҒбһү бһҳбһ·бһ“бһ”бҹ’бһҠбһјбһҡ canvas, бһ–бһ»бһҳбҹ’бһ–бһўбһҖбҹ’бһҹбһҡ, бһҸбҹҶбһ”бһ“бҹӢбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶ бһ¬ user-agent бһЎбһҫбһҷ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһ”бҹ’бһҠбһјбһҡ IP бһ бһҫбһҷбһҹбҹ’бһ“бһ¶бһҳбһҳбҹ’бһҡбһ¶бһҳбһҠбҹғбһ“бҹ…бһҠбһҠбҹӮбһӣбҹ”
