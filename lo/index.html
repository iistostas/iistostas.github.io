ເຄື່ອງມືກວດລາຍນິ້ວມືໂປຣແກຣມທ່ອງເວັບ

ກົດປຸ່ມເພື່ອອ່ານສັນຍານລາຍນິ້ວມືທີ່ໂປຣແກຣມທ່ອງເວັບຂອງທ່ານເປີດເຜີຍຢ່າງງຽບໆ — canvas, WebGL, ຟອນທີ່ຕິດຕັ້ງ, ເຂດເວລາ, ໜ້າຈໍ ແລະ User-Agent — ຈາກນັ້ນໄດ້ລະຫັດລາຍນິ້ວມືລວມ ແລະ ເບິ່ງວ່າແຕ່ລະສັນຍານລະບຸຕົວທ່ານໄດ້ງ່າຍປານໃດ. ທຸກຢ່າງຄຳນວນໃນເຄື່ອງຂອງທ່ານ ບໍ່ມີຂໍ້ມູນຖືກອັບໂຫລດ.

ເຄື່ອງມືນີ້ເຮັດວຽກທັງໝົດໃນໂປຣແກຣມທ່ອງເວັບຂອງທ່ານ. ບໍ່ອັບໂຫລດ ຫຼື ເກັບຂໍ້ມູນໃດໆ.

ສັນຍານທີ່ປະກອບເປັນລາຍນິ້ວມືໂປຣແກຣມທ່ອງເວັບ

ເວັບໄຊບໍ່ຕ້ອງການ cookie: ພຽງເກັບກຳຄຳຕອບທີ່ເບິ່ງຄືບໍ່ມີໄພຂອງໂປຣແກຣມທ່ອງເວັບ ກໍສ້າງຕົວລະບຸທີ່ຄ່ອນຂ້າງໝັ້ນຄົງໄດ້. ສັນຍານທົ່ວໄປລວມມີຄວາມແຕກຕ່າງເລັກນ້ອຍຂອງ GPU ແລະ driver ເມື່ອສະແດງ canvas/WebGL, ບັນຊີຟອນທີ່ຕິດຕັ້ງ, ເຂດເວລາ, ຄວາມລະອຽດໜ້າຈໍ ແລະ ຄວາມເລິກສີ, ພາສາ, ຈຳນວນແກນ CPU, ໜ່ວຍຄວາມຈຳອຸປະກອນ ແລະ ການຮອງຮັບການແຕະ. ແຕ່ລະຄ່າບໍ່ເປັນເອກະລັກ ແຕ່ເມື່ອລວມກັນມັກກາຍເປັນເອກະລັກ.

ແຕ່ລະສັນຍານລະບຸຕົວໄດ້ແຮງປານໃດ (entropy)

Entropy ວັດແທກວ່າສັນຍານໜຶ່ງໃຫ້ຂໍ້ມູນລະບຸຕົວຫຼາຍປານໃດ (ໜ່ວຍ bit — ຍິ່ງສູງຍິ່ງແຍກຄົນໄດ້ດີ). Panopticlick ຂອງ EFF ວັດໄດ້ສະເລ່ຍປະມານ 18.1 bit ໂດຍລາຍນິ້ວມືປະມານ 83.6% ເປັນເອກະລັກ; AmIUnique (Laperdrix et al., 2016) ພົບປະມານ 89.4% ເປັນເອກະລັກໃນ 118,000 ຕົວຢ່າງ. ຕາຕະລາງລຸ່ມນີ້ໃຫ້ການປະເມີນຄ່າຄ່າວໆຕໍ່ສັນຍານທີ່ສັງລວມຈາກການຄົ້ນຄວ້າສາທາລະນະນັ້ນ.

Entropy ລາຍນິ້ວມືຕາມສັນຍານ (ການປະເມີນ 2026)

ຕໍ່ໄປນີ້ແມ່ນການປະເມີນປີ 2026 ທີ່ສັງລວມຈາກການຄົ້ນຄວ້າລາຍນິ້ວມືສາທາລະນະ (EFF Panopticlick 2010, AmIUnique 2016) ສະແດງປະລິມານຂໍ້ມູນລະບຸຕົວໂດຍປະມານຕໍ່ສັນຍານ: ການປະເມີນ ບໍ່ແມ່ນການວັດໂດຍກົງ ສຳລັບອ້າງອີງເທົ່ານັ້ນ.

ຄຳຖາມທີ່ພົບເລື້ອຍ

ລາຍນິ້ວມືໂປຣແກຣມທ່ອງເວັບແມ່ນຫຍັງ?

ລາຍນິ້ວມືໂປຣແກຣມທ່ອງເວັບແມ່ນຕົວລະບຸທີ່ສ້າງຈາກການລວມຄຸນສົມບັດທີ່ໂປຣແກຣມທ່ອງເວັບ ແລະ ອຸປະກອນຂອງທ່ານເປີດເຜີຍ (user agent, ໜ້າຈໍ, ເຂດເວລາ, ຟອນ, ການສະແດງ canvas/WebGL ແລະ ອື່ນໆ). ເຖິງບໍ່ມີ cookie ເວັບໄຊກໍຈື່ໂປຣແກຣມທ່ອງເວັບອັນດຽວກັນຂ້າມການເຂົ້າຊົມໄດ້.

ເຄື່ອງມືກວດນີ້ປອດໄພບໍ? ມັນອັບໂຫລດຂໍ້ມູນຂອງຂ້ອຍບໍ?

ບໍ່. ການອ່ານ ແລະ ການຄຳນວນທັງໝົດເກີດໃນໂປຣແກຣມທ່ອງເວັບດ້ວຍ JavaScript ຜົນລັບສະແດງສະເພາະໜ້ານີ້ ບໍ່ສົ່ງໄປເຊີບເວີໃດໆ ແລະ ບໍ່ເກັບໄວ້.

ເປັນຫຍັງລາຍນິ້ວມືຂອງຂ້ອຍຈຶ່ງເປັນເອກະລັກ?

ຄຸນສົມບັດດຽວ (ເຊັ່ນ ເຂດເວລາ) ມີຄົນຄືກັນຫຼາຍ ແຕ່ຄຸນສົມບັດຫຼາຍສິບອັນລວມກັນສ້າງພື້ນທີ່ຄ່າໃຫຍ່ຫຼວງ. ການຄົ້ນຄວ້າສາທາລະນະສະແດງວ່າໂປຣແກຣມທ່ອງເວັບສ່ວນໃຫຍ່ມີລາຍນິ້ວມືລວມເປັນເອກະລັກ ຈຶ່ງຖືກຕິດຕາມຂ້າມເວັບໄຊໂດຍບໍ່ຕ້ອງເຂົ້າສູ່ລະບົບ.

ລາຍນິ້ວມື canvas ແມ່ນຫຍັງ?

ໂປຣແກຣມທ່ອງເວັບແຕ້ມຂໍ້ຄວາມ ຫຼື ຮູບອັນດຽວກັນລົງໃນ canvas ທີ່ເຊື່ອງໄວ້ ແລ້ວອ່ານ pixel ກັບຄືນ. GPU, driver, ຟອນລະບົບ ແລະ anti-aliasing ທີ່ຕ່າງກັນສ້າງຄວາມແຕກຕ່າງເລັກແຕ່ໝັ້ນຄົງ ເຮັດໃຫ້ canvas ເປັນສັນຍານທີ່ລະບຸຕົວແຮງທີ່ສຸດອັນໜຶ່ງ.

ໂໝດ incognito ເຊື່ອງລາຍນິ້ວມືໄດ້ບໍ?

ສ່ວນຫຼາຍບໍ່ໄດ້. ໂໝດສ່ວນຕົວລຶບ cookie ແລະ ປະຫວັດໃນເຄື່ອງເປັນຫຼັກ ແຕ່ canvas, WebGL, ຟອນ, ເຂດເວລາ ຍັງເປີດເຜີຍຄືເກົ່າ ດັ່ງນັ້ນພຽງໂໝດສ່ວນຕົວບໍ່ສາມາດຫຼົບການຕິດຕາມລາຍນິ້ວມືໄດ້.

ຈະຫຼຸດການຖືກຕິດຕາມດ້ວຍລາຍນິ້ວມືແນວໃດ?

ທ່ານສາມາດຫຼຸດການເປີດເຜີຍ (ໃຊ້ການຕັ້ງຄ່າທົ່ວໄປ ປິດບາງ API) ຫຼື ໃຊ້ໂປຣແກຣມທ່ອງເວັບປ້ອງກັນການກວດຫາສຳລັບຫຼາຍບັນຊີ — ມັນກຳນົດລາຍນິ້ວມືແຍກກັນ ຄວບຄຸມໄດ້ ແລະ ສອດຄ່ອງໃຫ້ແຕ່ລະ profile ເພື່ອບໍ່ໃຫ້ບັນຊີເຊື່ອມໂຍງກັນ.

VPN ປ່ຽນລາຍນິ້ວມືຂອງຂ້ອຍໄດ້ບໍ?

ບໍ່ໄດ້. VPN ປ່ຽນພຽງ IP ຂາອອກ ບໍ່ປ່ຽນ canvas, ຟອນ, ເຂດເວລາ ຫຼື user-agent ດັ່ງນັ້ນປ່ຽນ IP ແລ້ວລາຍນິ້ວມືກໍຍັງຄືເກົ່າ.
